Welcome to raylib!

raylib is an simple and easy to use library for enjoying video games development.

It's written in C, build on top of OpenGL, custom GLFW, miniaudio, stb... etc.

It's API is simple yet powerful, allowing us to create any kind of game and application!

Example game with raylib. 


						#include "raylib.h"

						int main(int ac, char **av) {
							InitWindow(800, 600, "Hello, raylib!");
							InitAudioDevice();

							while (!WindowShouldClose()) {
								/* update here */

								BeginDrawing();
								ClearBackground(WHITE);

								/* render here */

								EndDrawing();
							}

							CloseAudioDevice();
							CloseWindow();
							return (0);
						}
